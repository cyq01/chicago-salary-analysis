# Chicago Salary Analysis

## Overview
Analyzed ~32k City of Chicago employee records to understand salary patterns and drivers of compensation.

## Approach
- Data cleaning & EDA  
- Feature engineering (job levels)  
- Linear regression model  

## Key Findings
- Salary varies significantly across departments  
- Department impact > job level impact  
- Executive roles earn the highest compensation  

## Tools
Python, Pandas, Linear Regression
